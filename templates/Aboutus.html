About Us

The Personalized News Feed web application is designed to provide a tailored news experience to users, ensuring they receive only the content that interests them, based on their selected preferences. This application aims to minimize information overload, enabling users to easily access relevant and up-to-date news without the need to browse through irrelevant articles. This web application Consists of two role one is admin another is user. Admin has power to add news related information in this application. Admin can upload text related news information based on category. Initially users need to register with the application by submitting their details through web application. After that user can login this application using their username and password they can select the news category after successful selection user can able to view complete news information based on selection. This project is aimed at develop is to minimize the manual process, without making them to take long time effectively allows user to get complete News information through web application. This web application has been developed with an intention to make the system user-friendly thus reducing the manual work. The application has been developed with advanced features.
